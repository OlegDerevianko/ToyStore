
<h1>ToyStore - Free Ecommerce Template</h1>
<p>A modern, colorful, and fully responsive ecommerce website template for a toy store. Built with HTML, CSS, and featuring a clean design with smooth animations, mobile-friendly navigation, and interactive elements.</p>
<h2>📋 Features</h2>
<ul>
  <li> 
    <h3>Fully Responsive Design</h3> 
    <p>Perfectly optimized for mobile, tablet, and desktop devices</p> 
  </li> 
  <li> 
    <h3>Mobile Hamburger Menu</h3> 
    <p>Smooth sliding menu with overlay for mobile devices</p> 
  </li> 
  <li> 
    <h3>Interactive Product Cards</h3> 
    <p>Hover effects with animations and transitions</p> 
  </li> 
  <li> 
    <h3>Category Showcases</h3> 
    <p>Stuffed animals and wooden toys sections with product grids</p> 
  </li> 
  <li> 
    <h3>Shopping Cart Integration</h3> 
    <p>Cart counter with visual feedback</p> 
  </li> 
  <li> 
    <h3>Social Media Integration</h3> 
    <p>Links to social platforms with hover animations</p> 
  </li> 
  <li> 
    <h3>Newsletter Subscription</h3> 
    <p>Email subscription form with stylish design</p> 
  </li> 
  <li> 
    <h3>Instagram Gallery</h3> 
    <p>Responsive image grid showcasing products</p> 
  </li> 
  <li> 
    <h3>Clean Animations</h3> 
    <p>Smooth hover effects, transitions, and micro-interactions</p> 
  </li> 
  <li> 
    <h3>Accessibility Focused</h3> 
    <p>Semantic HTML and ARIA labels for better accessibility</p> 
  </li> 
</ul>

<ul> 
  <li>
